# 📈 積立投資シミュレーター（kabubeginner-app）

投資に興味はあるけれど「むずかしそう」「こわい」と感じている初心者のための、**積立投資シミュレーター**です。
毎月の積立額と、3つの質問に答えるかんたん診断だけで、将来の資産がどう増えていくかをグラフで確認できます。

🔗 **公開URL**: https://kenji0115.github.io/kabubeginner-app/

---

## ✨ 特長

- **3ステップのかんたん診断** — 質問に答えるだけで、資産配分と想定年率が自動で決まります
- **元本と運用益をグラフで可視化** — 年ごとの積み上げ棒グラフで、お金の育ち方がひと目でわかります
- **複利の実感** — 運用益が元本を上回る「複利が加速する年」をハイライト
- **初心者にやさしい設計** — むずかしい入力は不要。プリセットや目盛り付きスライダーで直感的に
- **正直なリスク表示** — こわがらせすぎず、軽く考えさせすぎない、ちょうどいいリスク説明
- **用語ミニ解説** — インデックス／複利／分散投資などを、その場でさっと確認
- **結果のシェア** — 条件をURLで共有、結果を画像で保存

---

## 🧭 使い方

1. **かんたん診断（任意）**
   3つの質問に順番に答えます。選ぶと**資産配分**と**想定年率**が決まります。
   - **ステップ1**　安定志向（インデックス）／積極志向（アクティブ）
   - **ステップ2**　守りより／バランス／攻めより（株式と債券の比率）
   - **ステップ3**　米国中心／日本も厚め／全世界株（オルカン）
   > 「やり直す」でいつでも選び直せます。

2. **毎月の積立金額を決める**
   プリセット（5,000〜50,000円）から選ぶか、自由に入力します（上限5万円）。

3. **積立期間を決める**
   スライダーで1〜50年から選びます。

4. **結果を見る**
   積立元本・運用益・最終資産と、年ごとのグラフが表示されます。
   「🔗 条件をコピー」でURL共有、「📷 画像を保存」でPNG保存ができます。

## 🔢 計算ロジック

### 資産の増え方（複利）
毎月の積立に対して「月利 ＝ 年率 ÷ 12」で毎月複利運用したと仮定し、年ごとの評価額を計算します。
（実装: [`src/calc.js`](src/calc.js) の `simulate()`）

### 診断による資産配分と想定年率
- **株式・債券の比率**はステップ2（攻め・守り）で決まります（守り 株式40% ／ バランス 60% ／ 攻め 90%）。
- **想定年率**は株式の地域（ステップ3）で変わります（米国 ＞ 全世界 ＞ 日本）。
- 想定年率の根拠は、**GPIF（年金積立金管理運用独立行政法人）が公表する期待リターン**（外国株式 ≈ 7.2% / 国内株式 ≈ 5.6% / 国内債券 ≈ 0.7%）を参考に丸めた概算値です。

> ⚠️ 本シミュレーションはあくまで概算です。将来の運用成果を保証するものではありません。

## 💻 ローカルでの開発

### 必要環境
- Node.js 20 以上

### セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動（http://localhost:5173/kabubeginner-app/）
npm run dev

# テストの実行
npm test

# 本番ビルド（dist/ に出力）
npm run build

# ビルド結果のプレビュー
npm run preview
```

## 🚀 デプロイ

`main` ブランチに push すると、GitHub Actions が自動で **テスト → ビルド → GitHub Pages へのデプロイ** を実行します。
（設定: [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml)）

初回のみ、リポジトリの **Settings → Pages → Build and deployment → Source** を **「GitHub Actions」** に設定する必要があります。
